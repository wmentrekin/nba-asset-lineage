  ## 01-sankey-sports-timeline.png
  Prompt: Design a polished longform sports data visualization for a Memphis Grizzlies asset-lineage story page. The main graphic is a very wide horizontal timeline covering 10 years, with one day as the underlying time unit. Assets appear as continuous flowing strands, similar to a Sankey diagram mixed with a transaction graph. The top lanes contain the most important players, lower lanes contain less prominent roster assets, two-way players, and future draft picks. At transaction moments, nearby strands converge into clean junctions and then diverge into new lane positions. When outgoing assets leave, the remaining strands compact upward and incoming assets enter from lower slots. The page feels like a premium editorial interactive, minimal narrative text, graph-first, highly readable, elegant sports color palette, clean typography, subtle axis ticks, serious and modern, not cartoonish, not dashboard-like.
  Prompt goal: Sankey-like asset graph
  What feels right: The visual flow of the asset lines in general, grouping core players at the top, then role players, then 2 way then draft picks, color scheme using team colors
  What feels wrong: The representaiton of draft capital, each pick should be its own line
