# Validation Checklist

Use this checklist in `$validate`.

- [ ] Implemented scope matches approved plan
- [ ] Acceptance criteria were checked end to end
- [ ] Review outcome is acceptable
- [ ] Test outcome is acceptable
- [ ] Known blockers are resolved or explicitly accepted
- [ ] Pre-commit checks or equivalent final checks were run
- [ ] Remaining caveats are documented
- [ ] Final readiness decision is explicit
